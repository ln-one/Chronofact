# Deployments

Deployment scripts write local chain outputs here.

Expected generated files:

- `artifacts/ChronofactRegistry.json`: Solidity ABI and bytecode.
- `ganache/ChronofactRegistry.json`: local Ganache deployment address, tx hash,
  receipt summary, network, and ABI.

Generated deployment outputs are local demo artifacts. Review them before
committing because addresses and transaction hashes are environment-specific.
